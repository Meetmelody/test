LAMP = Linux/Windows + Apache + MySQL + PHP/Python/Perl，Web Server 用 Apache，数据库用 MySQL，服务器脚本用PHP
J2EE，Java 世界的架构，通常是企业用的，常见还会搭配一种 UNIX 做操作系统，Web Server用Apache，Tomcat 转换 JSP 到 Java 给服务器程序用（其实它也自带 Web Server），Oracle 数据库等等。淘宝现在也是从LAMP转型到了J2EE。
微软，通常会搭配 Windows Server 操作系统，SQL Server 数据库， Web Server用IIS。StackOverflow和京东（曾经）就是这个架构。
MEAN，MongoDB做数据库，Express做 Web Framework，Angular 做前端的 JavaScript 框架，Node.js 用于编写 Web Server。
